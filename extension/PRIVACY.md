# Scout extension privacy summary

Scout installs an idle page bridge so it can work reliably on public job websites. The bridge does not collect or transmit anything until the member opens Scout. Scout then reads the title, company, location, and description from that active tab only. The member can review and edit every field before sending it.

When sent, the job URL and reviewed job details are stored in the member’s Scout account and shared with their assigned Human Assistant for application preparation. Scout stores the sign-in session locally in Chrome and remembers the selected job profile per website. It does not sell browser data, track unrelated browsing, or load remote executable code.

Members can disconnect the extension from its menu and can request account deletion through Scout.
